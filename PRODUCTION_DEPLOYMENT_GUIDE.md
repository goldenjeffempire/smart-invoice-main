## âš™ï¸ RENDER.YAML CONFIGURATION

```yaml
services:
  - type: web
    name: smart-invoice
    env: python
    plan: free
    buildCommand: pip install -r requirements.txt && npm install && npm run build:css && python manage.py migrate && python manage.py collectstatic --noinput
    startCommand: gunicorn smart_invoice.wsgi:application --bind 0.0.0.0:$PORT --workers 2 --timeout 30
    healthCheckPath: /health/
    envVars:
      - key: DEBUG
        value: false
      - key: SECRET_KEY
        generateValue: true
      - key: ALLOWED_HOSTS
        value: smart-invoice.onrender.com
      - key: DATABASE_URL
        fromDatabase:
          name: smartinvoice
          property: connectionString
      - key: EMAIL_HOST
        value: smtp.sendgrid.net
      - key: EMAIL_HOST_PASSWORD
        sync: false  # Must be set manually
    databases:
      - name: smartinvoice
        plan: free
        version: "14"
```

## ðŸ†˜ TROUBLESHOOTING

### Common Issues After Deployment

**Issue:** Pages show unstyled or dark mode not working
- **Solution:** Clear browser cache, hard refresh (Ctrl+Shift+R)

**Issue:** Database connection errors
- **Solution:** Verify DATABASE_URL environment variable

**Issue:** Email not sending
- **Solution:** Verify EMAIL_HOST_PASSWORD is set correctly

**Issue:** Static files returning 404
- **Solution:** Ensure `collectstatic` ran during build

**Issue:** High memory usage
- **Solution:** Adjust gunicorn workers in render.yaml
